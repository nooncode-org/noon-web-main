## The 5 reference About pages — what to learn from each

Owner's chosen bar: **OpenAI · Anthropic · Vercel · Linear · Cursor**. Analyzed 2026-06-29.

### Vercel — https://vercel.com/about
- **Structure:** conviction hero → investor logo grid → named individual-investor cards → careers CTA. Very lean.
- **Hero:** *"Vercel frees the world to ship what's next."* (a belief, not a description.)
- **Trust device:** credibility by *who believes in us* (VCs + notable founders), not metrics.
- **Take for Noon:** open with conviction, not "a technology development company…". Lean is fine. (Noon has no investors → skip that device; use real work instead.)

### OpenAI — https://openai.com/about/
- **Structure:** mission-led → "Our structure" → leadership → careers-as-mission.
- **Mission:** *"ensure that artificial general intelligence benefits all of humanity."* Everything hangs off one big mission.
- **Trust device:** "Our structure" (how they're organized to deliver the mission) + leadership named.
- **Take for Noon:** anchor the page to one clear belief; a "how we're built to guarantee X" beat (for Noon: the human-review gate / ownership) is a strong trust device.

### Anthropic — https://www.anthropic.com/company
- **Structure:** belief hero → purpose → "we build safer systems" → "safety is a science" → interdisciplinary → team (4 functions) → **7 numbered values** → governance.
- **Hero:** *"Making AI systems you can rely on."*
- **Devices:** a **numbered values manifesto**; governance as trust; team-by-function; voice with character (*"high-trust, low-ego"*).
- **Take for Noon:** turn "operating principles" into a real **values manifesto**; alternate distinct block shapes; let the voice have a stance.

### Linear — https://linear.app/about
- **Structure:** hero (product video) + "the moment" POV → "new species of product tool" → **team** (3 founders + 100+ names) → **backers** (VCs + founder investors) → CTA.
- **Hero:** *"Building tools for the next era of product development."* POV: *"Software development is at an inflection point."*
- **Devices:** founder cards, 100-name grid, logo social proof, an identity line — *"We are designers and engineers. Problem solvers and storytellers… makers at heart."*
- **Trust:** *"Founded in 2019… tool of choice for more than 33,000 companies"* + named customers (OpenAI, Coinbase, Ramp).
- **Take for Noon:** **show the people** (the biggest gap for a "human judgment" company); an identity line; a "the moment" POV section; a scale/founding stat.

### Cursor — https://cursor.com/enterprise
- **Structure (richest rhythm):** hero → testimonial+demo pairs → **stat band** → control features → models → bugbot → **trust/security grid** → **executive testimonials** → **case studies** → FAQ → CTA. Every section a different block type with a real visual.
- **Hero:** *"Develop enduring software at scale."*
- **Devices:** hard **stats** (64% Fortune 500 · 93% preference · 100M lines/day), exec testimonials, security/compliance grid, case-study cards, FAQ.
- **Voice:** *"not a convenience add-on; it's a scale-multiplier for the whole org."*
- **Take for Noon:** stats band, selected-work case cards, a dedicated **ownership/security/trust** beat, section-level CTAs, FAQ (already have).

## Cross-cutting visual-structure lessons (how they stay dynamic/editorial/premium, *not* a list)
1. **No two adjacent blocks share a shape** — alternate: full-width statement ↔ card grid ↔ 2-col text+UI ↔ stat band ↔ logo wall ↔ testimonial+demo.
2. **Every section *shows*, not just tells** — paired with a real visual: product UI / mockup, diagram (node-graph), team photos, stat band.
3. **Density rhythm** — big airy hero → dense proof grid → airy statement → dense cards. Whitespace as punctuation.
4. **Social-proof devices** — logo walls, named-people cards, hard stats (pick Noon's *honest* set: real work + the review gate + ownership).
5. **Section-level CTAs** sprinkled, not only at the end ("Learn more →").
6. **A story arc:** belief → why/moment → who → proof → how → join.
